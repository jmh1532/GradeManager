# GradeManager
수강신청 , 수강신청 내역을 출력하는 간단한 형태의 웹페이지

## Running the tests

1. 아두이노 불꽃센서와 온도감지센서를 통한 화재감지(아두이노)
```
아두이노 화재감지 위치 -> 웹서버, 화재위치 DB에 저장
```
